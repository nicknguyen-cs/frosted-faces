# Agent Instructions

This repository is an [Agent Skill](https://agentskills.io/). Read [SKILL.md](SKILL.md) for complete documentation on integrating Contentstack Personalize with Google Tag Manager via the GTM API v2.

SKILL.md contains:
- GTM API v2 authentication and setup
- Programmatic creation of Personalize tags (Initialize, Trigger Impressions, Trigger Event, Set Attributes, Set UserId)
- Trigger creation patterns (Page View, Click, Custom Event)
- Tag sequencing and firing options
- Code-side integration (Next.js GTM snippet, dataLayer pushes)
- Publishing workflow
