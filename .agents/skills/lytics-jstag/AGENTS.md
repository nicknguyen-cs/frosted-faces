# Agent Instructions

This repository is an [Agent Skill](https://agentskills.io/). Read [SKILL.md](SKILL.md) for complete documentation on the Lytics JavaScript Tag (jstag) — behavioral tracking, identity resolution, profile access, content recommendations, and declarative `lx-*` attribute tracking.

SKILL.md contains:
- Installation (script tag, GTM, self-hosted)
- Configuration options (streams, cookies, entity identifiers)
- `jstag.send()` for custom events and identity
- `jstag.pageView()` for SPA route changes
- Declarative `lx-*` HTML attribute tracking
- Visitor profile access via `entityReady` callback
- Content recommendations via `jstag.recommend()`
- Integration patterns (GA4, Sitecore, LiveRamp)
- Common tagging patterns (login, e-commerce, forms)
- Troubleshooting and debugging commands
